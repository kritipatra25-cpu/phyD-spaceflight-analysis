# Biomni Integrated Analysis

This directory contains the final integrated systems biology analysis of Arabidopsis germination under deep-space related stress conditions.

## Datasets Integrated

* OSD-120 (Root Microgravity)
* OSD-678 (Shoot Microgravity)
* OSD-658 (Galactic Cosmic Radiation)
* Null Magnetic Field datasets derived from Maffei et al.

## Main Deliverables

### unified_germination_model.pdf

Primary integrated systems biology model describing how microgravity, radiation, and magnetic-field perturbations influence germination-related biological processes.

### integrated_systems_biology_report.pdf

Detailed systems-level analysis integrating transcriptomic, pathway, tissue, and cell-type evidence.

### nmf_systems_biology_summary.pdf

Focused analysis of Null Magnetic Field responses and comparison with spaceflight datasets.

### cross_dataset_enrichment_summary.pdf

Cross-study enrichment analysis identifying biological programs that are reproducibly affected across independent datasets.

## Methods

* Differential expression analysis
* Heatmap-based pathway exploration
* KEGG pathway modelling
* SBGN pathway modelling
* Single-cell atlas projection
* Tissue-specific systems biology modelling
* Cross-dataset enrichment analysis

## Major Findings

* Vascular developmental programs emerge as the most consistently perturbed biological module across multiple space-related stressors.
* Hypocotyl-associated programs show context-dependent behavior.
* Microgravity, Galactic Cosmic Radiation, and Null Magnetic Field conditions exhibit both shared and distinct regulatory responses.

## Repository Context

The analyses contained in this folder represent the final synthesis of all supporting analyses present elsewhere in the repository.
