This folder contains **SBGNView-based systems biology models** of the PhyD gene regulatory network.

These diagrams represent the **advanced stage of pathway modeling**, improving upon KEGG-based representations by using standardized Systems Biology Graphical Notation (SBGN).

The SBGN models are used to:

- Represent molecular interactions in a structured and standardized format
- Improve clarity of regulatory relationships compared to KEGG diagrams
- Capture mechanistic details of gene regulation networks

  Why SBGNView?

Compared to KEGG pathway diagrams:

- KEGG → provides simplified pathway overview
- SBGNView → provides **standardized, detailed mechanistic representation**

  This folder includes:

- SBGN pathway diagrams (PNG/SVG formats)
- Advanced regulatory network visualizations
- Condition-specific or refined pathway models

  These models were generated using SBGNView-based scripts in R.

  These data are obtained from OSD 120: https://osdr.nasa.gov/bio/repo/data/studies/OSD-120
