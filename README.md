## Final Integrated Systems Biology Analysis

The final synthesis of this project is located in the `Biomni/` directory.

This analysis integrates:

* OSD-120 (Arabidopsis root microgravity)
* OSD-678 (Arabidopsis shoot microgravity)
* OSD-658 (Galactic Cosmic Radiation exposure)
* Null Magnetic Field (Maffei et al.) datasets

The Biomni analysis combines:

* Differential expression analyses
* Heatmap-based pathway investigations
* KEGG pathway modelling
* SBGN pathway modelling
* Single-cell atlas projection
* Tissue-specific systems biology modelling
* Cross-dataset integration

The folders outside the Biomni directory represent the intermediate analyses, visualizations, pathway studies, and computational workflows that contributed to the final integrated systems biology model.

For readers interested in the primary project conclusions, begin with:

`Biomni/unified_germination_model.pdf`

and

`Biomni/integrated_systems_biology_report.pdf`
