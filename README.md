Project Overview

This repository contains transcriptomic analyses investigating the effects of altered magnetic and spaceflight environments on Arabidopsis thaliana.

The project compares:

NASA GeneLab OSD-120 (Spaceflight vs Ground Control)
NASA GeneLab OSD-678 / GLDS-612 (Spaceflight vs Ground Control)
Published Near-Zero Magnetic Field (NMF) datasets obtained from supplementary files of peer-reviewed research articles

The primary focus of this work is to investigate potential relationships between:

Cryptochrome (CRY) signalling
Circadian regulation
Reactive Oxygen Species (ROS) signalling
Gibberellin metabolism and signalling
Photosynthesis
Photorespiration

and determine whether microgravity reproduces molecular signatures observed under Near-Zero Magnetic Field (NMF) conditions.

📌 Overview
This repository contains a systems biology analysis of the **PhyD gene regulatory network** under different biological conditions.

The project focuses on constructing and comparing gene interaction networks using:
- KEGG pathway-based visualization (baseline model)
- SBGNView-based systems biology graphical modeling (advanced model)

The goal is to move from a simplified pathway representation to a more structured and mechanistic systems biology framework.

🧬 Biological Focus
PhyD (Phytochrome D) is involved in:
- Light signaling pathways
- Circadian rhythm regulation
- Plant development and flowering time regulation

This project explores how PhyD interacts with downstream regulatory networks

🧪 Methodology
1. KEGG Pathway Modeling (Baseline)
- Gene interactions mapped using KEGG pathway database
- Visualization generated using pathway analysis tools
  
2. SBGNView Modeling (Advanced)
- Standardized Systems Biology Graphical Notation (SBGN)
- Structured representation of molecular interactions
- Improved clarity of regulatory relationships

3. Heatmaps
